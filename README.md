# v6-image-validator
Scripts and lists to support investigation of node Vantage6 images

> [!WARNING]
> `known_digests_from_github.csv` is not yet an exhaustive list of images from
> the last 90 days (since January 26, 2026).

Scripts under `./pre-review-in-dev/` are in development and haven't been reviewed yet.

It's advisable to analyze a potentially compromised VM after placing it in
isolation and with a full backup available for potential post-cleanup analysis.
